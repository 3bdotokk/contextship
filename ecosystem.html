README.md

@contextship/router

The Context Router is the core routing engine that determines which context reaches which agent. It supports deterministic, semantic-nearest, load-balanced, priority-queue, and broadcast strategies.

Build Passing npm v2.4.0 Coverage 94%

Installation

npm install @contextship/router

Quick Start

import { Router, Strategy } from '@contextship/router';

const router = new Router({
  strategy: Strategy.DETERMINISTIC,
  fallback: Strategy.SEMANTIC_NEAREST,
  maxLatency: '10ms',
  cache: { enabled: true, ttl: '5m' }
});

const result = await router.route({
  target: 'finance-ai',
  query: 'Analyze Q4 risk exposure',
  sources: ['market-data', 'risk-models']
});

console.log(result.latency); // '7ms'
console.log(result.score);   // 0.94

@contextship/memory

Production-grade memory infrastructure for AI agents. Implements working, episodic, and semantic memory tiers with configurable consistency, vector search, and automatic lifecycle management.

Build Passing npm v2.3.1 Coverage 91%

Installation

npm install @contextship/memory

Quick Start

import { MemoryEngine, Tier } from '@contextship/memory';

const memory = new MemoryEngine({
  tiers: {
    working:  { capacity: '128MB', ttl: '15m' },
    episodic: { capacity: '2GB',   ttl: '7d' },
    semantic: { capacity: 'unlimited', ttl: 'persistent' }
  },
  vectorIndex: { dimensions: 1536, metric: 'cosine' }
});

// Store and recall
await memory.store({ content: 'User prefers ETFs', tier: Tier.SEMANTIC });
const results = await memory.recall({ query: 'investment preferences' });

Key Features

  • ✅ Three-tier memory hierarchy (working, episodic, semantic)
  • ✅ Vector search with cosine/dot-product/euclidean metrics
  • ✅ Cross-agent memory synchronization
  • ✅ Configurable consistency (eventual, strong, causal)
  • ✅ Automatic garbage collection and TTL enforcement
  • 🚧 Memory compression and deduplication

contextship-orchestration

Define complex multi-agent workflows as declarative pipeline graphs. Built-in support for checkpointing, rollback, parallel execution, and deterministic replay for debugging.

Build Passing PyPI v1.8.0 Coverage 89%

Installation

pip install contextship-orchestration

Quick Start

from contextship import Orchestrator, Pipeline

orchestrator = Orchestrator(api_key="cship_live_sk_...")

pipeline = Pipeline(
    name="portfolio-analysis",
    steps=[
        {"agent": "data-collector", "action": "gather"},
        {"agent": "risk-analyzer", "action": "assess", "depends_on": [0]},
        {"agent": "reporter", "action": "generate", "depends_on": [0, 1]},
    ],
    checkpoint=True
)

result = await orchestrator.execute(pipeline)
print(result.duration)  # 1.2s
print(result.steps_completed)  # 3

Context Protocol Specification

An open protocol that standardizes how AI agents discover capabilities, negotiate context requirements, and exchange semantic information. Designed to complement MCP with context-specific primitives.

Spec v1.2 RFC Status: Draft 12 Implementations

Protocol Overview

// Context Protocol message format
{
  "protocol": "contextship/v2",
  "type": "context.request",
  "source": {
    "agent": "finance-ai",
    "capabilities": ["analysis", "forecasting"]
  },
  "request": {
    "context_type": "market-data",
    "freshness": "5m",
    "max_tokens": 4096,
    "semantic_constraints": {
      "relevance": 0.85,
      "coherence": 0.90
    }
  }
}
